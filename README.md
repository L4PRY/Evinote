# Evinote
Notes board webapp made with sveltekit and postgres.

# setup devenv
prequisites:
- (podman)[https://podman.io]
- (node.js)[https://nodejs.org/en/download/]

### 1. install dependencies
```bash
npm install
```

### 2. start database
```bash
npm run db:start
```

### 3. push schema to database
```bash
npm run db:push
```

### 4. start dev server
```bash
npm run dev
```
